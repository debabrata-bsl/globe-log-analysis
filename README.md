🌐 Globe Log Analysis
A Python-based log analysis tool designed to parse, process, and visualize server logs for the GlobalKela platform to monitor system performance and user activity.

🚀 Getting Started
Follow these steps to set up the project and run the viewer application on your local machine.

1. Clone the Repository
First, clone the repository to your local system:

Bash
git clone https://github.com/debabrata-bsl/globe-log-analysis.git
cd globe-log-analysis
2. Setup the Viewer App
Navigate to the viewer-app directory to install dependencies and start the development server:

Bash
Move into the viewer-app folder
cd viewer-app

Install necessary packages
npm install

Start the development server
npm run dev
📁 Project Structure
scripts/: Contains Python scripts for log parsing and data processing.

viewer-app/: The frontend application used to visualize the processed log data.
